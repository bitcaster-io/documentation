Terminology
===========

These are main concepts in the Bitcaster domain, more detailed
terminology can be found in the :ref:`glossary`

------------


Organization
~~~~~~~~~~~~
Organization is the primary subject in Bitcaster, it refers to your Company or Community,
it is where all the Bitcaster users belong to.


Channel
~~~~~~~

A channel represent a way that :ref:`Application` can use to propagate their :ref:`Event`


Application
~~~~~~~~~~~

As :ref:`Organization` you can have multiple Applications.


Event
~~~~~

As :ref:`Organization` you can have multiple Applications.
