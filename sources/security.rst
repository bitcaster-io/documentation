Security
========


SUPERUSER
---------

Superuser can work on System settings and configure Authenetication, Dispatachers and core components
of the system


OWNERS
------

Gains full permission across the organization.
Can manage :ref:`application` and Channels as well as perform catastrophic operations such as removing elements.


ADMINS
------

Admin privileges on any Application of which they're a member.
They can create new teams and events, as well as subscribe users to specific event


SUBSCRIBERS
-----------

Can subscribe events they have allowed to access, change their profile and addresses
